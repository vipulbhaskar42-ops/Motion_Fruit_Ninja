# Motion Fruit Ninja

A Fruit Ninja-inspired game built from scratch using Python and Pygame.

## Features
- Multiple fruits
- Bomb mechanics
- Score & High Score
- Lives system
- Fruit slicing animation
- Juice particle effects
- Restart functionality

## Tech Stack
- Python
- Pygame

## Inspiration
Inspired by a Fruit Ninja game concept seen on LinkedIn. The game was implemented from scratch using my own code and features.
